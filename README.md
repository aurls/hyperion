# Rude Web App Boilerplate

Primitive React/Node.js setup for fast lunch of web app development.

#### Stack

* React;
* Redux Toolkit;
* Node.js;
* TypeScript;
* SCSS;
* Webpack.

#### Commands

* `npm run start` runs development;
* `npm run build` gets a build.
